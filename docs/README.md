## 核心起点

如果你是第一次来到这里，建议按这个顺序开始：

1. 阅读根目录 `README.md`
2. 阅读 `docs/README.md`，了解仓库结构
3. 进入 `gitee/yi-yu-core/README.md`，理解易宇社区的基本框架
4. 阅读 `gitee/yi-yu-whitepaper/东方系统科学计算范式白皮书（V1.10）`
5. 选择一个方向开始：预测、白皮书、AGI 协作
