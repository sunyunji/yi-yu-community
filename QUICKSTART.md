# 快速入门指南

欢迎来到**易宇社区**——全球首个开创"东西方融合文明新范式"的开源研究阵地！

## 你是谁？选择相应的入门路径

### 🔬 如果你是学者 / 研究者

**你想了解：** 易宇社区的学术价值和研究方向

**入门步骤：**
1. 阅读 [README.md](./README.md) 中的"核心理念"部分
2. 查看 [gitee/yi-yu-whitepaper/](./gitee/yi-yu-whitepaper/) 中的易宇白皮书
3. 了解 [gitee/yi-yu-prediction/](./gitee/yi-yu-prediction/) 中的实战案例
4. 在 [docs/README.md](./docs/README.md) 找到针对你学科的学习路线图

**核心文档：**
- `gitee/yi-yu-whitepaper/东方系统科学计算范式白皮书（V1.10）`
- `gitee/yi-yu-whitepaper/易宇观公理体系：东方科学范式的数学根基.md`
- `gitee/yi-yu-prediction/太极地磁量化全域方程.md`

**后续行动：**
- 在 Issues 中提出研究问题或方向建议
- 考虑发起或加入某个研究子方向
- 撰写论文或案例研究

---

### 💻 如果你是开发者 / 工程师

**你想了解：** 如何参与技术开发和软件实现

**入门步骤：**
1. 快速浏览 [README.md](./README.md)
2. 查看 [gitee/yi-yu-core/](./gitee/yi-yu-core/) 了解架构
3. 阅读 [CONTRIBUTING.md](./CONTRIBUTING.md) 中的技术贡献指南
4. 查看 [docs/README.md](./docs/README.md) 中的"技术方向"

**可以做什么：**
- 把数学模型转换为可运行的代码
- 开发预测算法或验证工具
- 优化现有的计算框架
- 构建可视化或应用界面

**后续行动：**
- 在 Issues 中认领技术任务
- 创建算法原型或 PoC
- Pull Request 提交代码

---

### 📚 如果你是学生 / 初学者

**你想了解：** 易宇社区是什么，如何学习并参与

**入门步骤：**
1. 阅读 [README.md](./README.md) 的全文
2. 看 [docs/README.md](./docs/README.md) 中的概览
3. 选择一个感兴趣的主题：
   - 对数学感兴趣？→ 学习 [易宇白皮书](./gitee/yi-yu-whitepaper/)
   - 对预测感兴趣？→ 研究 [预测报告](./gitee/yi-yu-prediction/)
   - 对 AI 感兴趣？→ 了解 [AGI 方向](./gitee/zhicang/)
4. 在 Issues 中提问

**学习资源：**
- 中文：所有文档都用中文编写，附带学术解释
- 英文：[README_EN.md](./README_EN.md) 和白皮书英文版本

**后续行动：**
- 整理学习笔记，写成 PR
- 提出学习过程中的问题
- 制作学习总结或教程

## 所有人都可以做的事情

无论你是什么角色，这些事情总是欢迎的：

1. **提交 Issue**
   - 报告发现的错误或不规范
   - 提出改进建议
   - 分享想法或问题

2. **点 Star**
   - 帮助项目获得更多关注
   - 支持我们的开源精神

3. **分享**
   - 告诉朋友和同事
   - 在社交媒体分享
   - 邀请相关领域的人参与

4. **反馈**
   - 使用过程中的任何建议
   - 文档的可读性改进
   - 新的合作想法
