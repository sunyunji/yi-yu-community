# 贡献指南

欢迎加入易宇社区！无论你是经验丰富的研究者，还是刚进入这个领域的新人，我们都希望你的声音能成为共建东方系统科学的重要力量。

## 你可以贡献的内容

- 文档整理、结构化和翻译
- 白皮书、研究报告、课题解析
- 预测模型、算法思路、验证框架
- 东方术数与现代科学的融合案例
- 学术资料、参考文献、实验记录
- 项目宣传、社区活动、协作方式

## 基本方式

1. 阅读 `README.md` 和 `docs/README.md`
2. 了解本仓库当前结构和核心内容
3. 创建 Issue：如果你有内容建议、问题，或希望参与某个方向
4. 提交 Pull Request：如果你想直接修改文档、补充资料、优化结构

## 如果你不懂 Git

- 也可以先提交 Issue，描述你希望改进的内容
- 我们会帮你把想法整理成文档或任务
- 你也可以把修改内容直接写在 Issue 里，由社区协助完成

## 文档规范建议

- 使用 Markdown 格式（`.md`）
- 文件名尽量简洁、描述明确
- 章节标题清晰，可读性强
- 保留中文核心表达，同时欢迎附加英文说明

## 社区原则

- 尊重、开放、共建
- 兼容中西、融通古今
- 理论优先，验证支持

欢迎你把这片东方系统科学的新范式带向全球。
