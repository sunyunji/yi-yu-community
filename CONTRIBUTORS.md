## 📋 如何被列入此名单？

### 代码与技术贡献

1. Fork 本仓库
2. 创建特性分支
3. 提交 Pull Request
4. 获得批准后自动列入

### 研究与文献贡献

1. 在 Issues 中提出研究提案
2. 完成研究与验证
3. 提交论文或报告
4. 获得社区确认后列入

### 翻译与文档

1. 申报翻译任务（创建 Issue）
2. 提交翻译 PR
3. 获得批准后列入

### 推广与社区

1. 在社交媒体或会议分享易宇社区
2. 邀请学者参与
3. 组织工作坊或论坛
4. 向我们报告（Issue 或邮件）
5. 获得确认后列入

### 资助或合作

1. 通过 SUPPORT.md 联系
2. 讨论合作方式与条款
3. 达成协议
4. 列入合作伙伴名单

## 许可与条款

所有贡献均遵循本仓库的开源许可证（如有）。

通过向易宇社区提交贡献，你同意：
1. 你的贡献是原创的或有适当许可
2. 你授权易宇社区使用与传播你的贡献
3. 你同意被列入贡献者名单（可选隐名）
