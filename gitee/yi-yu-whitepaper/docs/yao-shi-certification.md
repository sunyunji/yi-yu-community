## 什么是「爻师」？

「爻师」（Yao Shi），意为通晓卦爻之理、践行易宇之道的贡献者。

**不是头衔，是文明共建的印记。**

我们不授予「专家」，我们铭刻「贡献者」。

## 认证等级（64级，每卦一级）

| 等级 | 卦象 | 含义 | 认证标准 |
|------|------|------|----------|
| 初九 | 乾 | 启动之始 | 提交首个「易-宇」映射模型或修正文档 |
| 九二 | 见龙在田 | 初显锋芒 | 被采纳为社区标准模型 |
| 九三 | 君子终日 | 持续贡献 | 3次有效PR，或完成1个可运行模块 |
| 九四 | 或跃在渊 | 突破性贡献 | 提出新映射关系并获3人以上验证 |
| 九五 | 飞龙在天 | 领导者 | 主导1个子项目（如「五行AI」「卦象可视化」） |
| 上九 | 亢龙有悔 | 传承者 | 撰写《易宇观》教学材料或翻译英文版 |
| 六二 | 坤 | 含弘光大 | 贡献10次以上，且被引用≥5次 |
| 六三 | 含章可贞 | 沉潜修德 | 完成3个文档翻译或校对 |
| 六四 | 括囊无咎 | 谨慎精进 | 修复3个关键Bug或优化性能 |
| 六五 | 黄裳元吉 | 中正致和 | 协调跨领域协作，促成2个以上项目融合 |
| 上六 | 龙战于野 | 化育天地 | 完成64卦全部映射，并推动至arXiv |

> 注：每级对应一卦，按《周易》顺序递进，不可跳级。其余卦象由Gitee Bot自动记录。

## 如何成为爻师？

| 步骤 | 操作 |
|------|------|
| 1. 贡献 | 向 yi-yu-community 提交 PR（代码、文档、模型、翻译） |
| 2. 评审 | 由3名现有爻师或管理员审核通过 |
| 3. 认证 | 系统自动在您的Gitee主页显示「爻师·[卦象]」徽章 |
| 4. 铭刻 | 您的姓名与贡献永久载入《爻师年鉴》 |
