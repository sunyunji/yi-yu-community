

# 易宇观 · 以易洞宇，以宇证易

> 开源共建东方系统科学的认知新范式

---

## 📜 项目愿景

易宇观是一个开放的文化+科学开源项目，致力于探索《易经》卦象系统与现代科学计算之间的深层映射关系。我们相信，卦象不是古董，而是人类文明的一套古老而精妙的符号计算系统。

**核心理念**：《易》不是复古，而是在为人类文明重写操作系统。
