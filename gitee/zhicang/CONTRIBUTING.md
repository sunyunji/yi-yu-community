# 贡献指南

感谢你愿意为 易宇社区 贡献力量！

## 如何贡献

1. Fork 本仓库
2. 创建你的功能分支 (`git checkout -b feat/your-feature`)
3. 提交你的修改 (`git commit -m 'add some feature'`)
4. 推送到分支 (`git push origin feat/your-feature`)
5. 提交 Pull Request

## PR 规范

- PR 标题简洁清晰
- 描述修改内容与原因
- 保持排版统一、格式整洁
- 不添加无关内容

## 社区精神

- 尊重、友善、包容
- 理性讨论，共同成长
- 长期主义，共建共生

如有任何问题，欢迎在 Issue 或社区群内提出。
