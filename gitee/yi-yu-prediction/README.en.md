
# yi-yu-prediction
Prediction Research Project of Yi-Yu Community

## Introduction
This project is a practical application of the "Yi-Yu View" theoretical system. It conducts predictive research in multiple fields such as geology, climate, and social rhythms using a model that integrates Eastern Yi studies and modern science, forming engineering-level research reports and observation data.

## Core Model
Adopts the Yi-Yu holographic model of "1 Core + 3 Supports + 1 Evolution Chain":
- Core: The universe is holographic, with love as the origin
- Supports: Taiji Point - Double Fish Eyes - Central Palace Five
- Evolution Chain: The logic of space-time energy conduction from origin to phenomenon

## Directory Description
- `/reports`: Published prediction research reports (Markdown format)
- `/data`: Observation data and raw materials supporting the reports
- `/docs`: Documentation explaining the Yi-Yu View theoretical system
