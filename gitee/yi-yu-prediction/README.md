# yi-yu-prediction
易宇社区（Yi-Yu Community）预测研究项目

## 项目简介
本项目为「易宇观」理论体系的落地实践，以东方易学与现代科学融合的模型，开展地质、气候、社会节律等多领域的预测性研究，形成工程级的研究报告与观测数据。

## 核心模型
采用「1个核心+3大支撑+1条演化链」的易宇全息模型：
- 核心：宇宙全息，爱为原点
- 支撑：太极点-双鱼眼-中宫五
- 演化链：从本源到现象的时空能量传导逻辑

## 目录说明
- `/reports`：已发布的预测研究报告（Markdown格式）
- `/data`：支撑报告的观测数据与原始资料
- `/docs`：易宇观理论体系说明文档
