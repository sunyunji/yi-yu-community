# 支持与资助

易宇社区是一项面向全球的**东西方融合文明新范式**的公益研究事业，致力于让东方古代智慧与现代科学完整融合，成果造福天下。

## 我们的原则

- 东方术数必须公益
- 不能垄断，不能牟利
- 任何资助都应支持社区建设与研究发展
- 尊重传统智慧与现代科学的融合路径

## 我们需要的支持

- 基金资助与科研支持
- 项目建设与技术开发
- 文档整理与传播推广
- 国际合作与学术交流

## 如果你愿意支持

欢迎有意资助的个人或机构与我们联系，共同支持易宇社区的长期发展。

我们希望：

- 为社会提供公开可用的东方系统科学研究资源
- 为研究者提供一个共建、共享、验证的平台
- 将家庭与个人投入转化为公益价值

### 说明

目前项目以公益原则运行，任何资助都将用于：

- 资料整理与翻译
- 研究与实验验证
- 社区协作与开放平台搭建
- 项目推广与教育传播

如果你愿意帮助我们，请通过 Github Issue、电子邮件或其它联系方式告诉我们你希望支持的方向。
